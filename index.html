Predictive Flood Forecasting Model

Conducted extensive hydrological research to build advanced Intensity-Duration-Frequency (IDF) curves for a major river basin, enhancing flood prediction and infrastructure planning.

  • Analyzed over 12,000 rainfall records using SQL and Python, ensuring high data integrity.
  • Developed a predictive model by resampling historical data into sub-daily timeframes using R and Python.
  • Optimized model accuracy by 23% using Chi-square, $R^2$, and MSE metrics across five distributions.
  • Designed compelling IDF curve visualizations with Matplotlib & Seaborn for strategic insights.
